# 蠢く口蓋

## GM向け情報

古尾 力(フルオ チカラ)と古尾 境子(フルオ キョウコ)は仲の良い夫婦だった。そんな二人は一人の子古尾 路凪を授かり仲睦まじく過ごす明るい未来があると信じていた。しかし、路凪が産まれてしばらく経った頃、境子が代理ミュンヒハウゼン症候群を患い、路凪へ虐待が行われるようになった。当初、境子は病院へ行き治療を受けていたが治療は長引いた。そんな折、力の精神が限界となり境子へ暴行を働くようになる。
父と母のそんな悪意に強くひかれたイゴーロナクは路凪に憑依し祝福を与えた。祝福を受けた路凪は両親の頭を食し殺害した。今までずっと喧嘩してた両親と三人で寝れる事を嬉しく思っている。

## 導入

探索者たちは古尾家へ行くことから始まる。
その理由は探索者によって自由に設定して構わない。教師であれば古尾 路凪の担任であったことにしてもよいだろう。医師や薬剤師であれば古尾 境子を知っていてもよい。はたまた、近所の主婦で古尾家と関わりがあっても良いかもしれない。
導入を考えることが難しければ古尾 境子の主治医から最近病院に来ていないため、様子を見てきてほしいといった依頼があったことにする事もできるだろう。

すでに治りかけだが身体のあちこちに傷跡がある。母親による暴行によるものであり、適切な技能を使用することで何かしらの道具を使用したようなものだとわかるだろう。刃物やガラス片での切り傷や火傷が主だったもの。

ぼんやりとした明かりが室内を照らしている。外からでも感じた生ゴミのような腐敗臭がさらに鼻腔を刺激する。壁や床の至る所には赤黒いシミが付いており、それが血痕であると想像することは容易である。そして、君たちの後ろでギイっと音がなり扉がひとりでに閉まった(正気度喪失:0/1)。
扉はとても開きそうにない。イゴーロナクの力によって閉じ込められる。
部屋の奥、キッチンと寝室からは強烈な腐敗臭がする。

### 母親の死体

スマホはロックされている。

父親のスマホは適切な技能を使用するか、後述のPCを使用して解除する事ができる。父親のスマホにはいくつか通知がある。通知があることはロックを解除できていない状態でもわかって良いがメッセージの内容までは見れない。また、通知は職場からの電話や同僚からのメッセージが主だが、その中に女性からのメッセージがある。通知は1週間程度前からのようである。

### イゴーロナク戦

探索者とNPCが言い争いなどを行う(または一定時間経過)と路凪の身体が膨張し、イゴーロナクが顕現する。
路凪が探索者の一人の腕を掴む。その力はとても子どもの力とは思えなかった。その痛みに耐えきれず思わず振り払ってしまう。
振り払われた路凪は床に転げてしまう。その直後、ギチギチとまるで腕や脚が悲鳴を上げるような音を鳴らしながら四肢が蠢き筋肉が肥大化した。皮膚は青白く変色し張り裂けそうなほど膨れ上がっている。
元々あった可愛らしい顔は肩や首が膨れ上がり見えなくなっていた。そして、 それは君たちにだらしなくよだれを垂らす口がついた腕を伸ばしてきた(正気度喪失:1/2D10)。
イゴーロナクの手を破壊することでイゴーロナクは退散される。本来と用途は異なるが本シナリオではその役割も持つことにする。イゴーロナクの手の耐久度は6程度とする(難易度調整は行ってよい)。
また、この地点で玄関の扉は開いており、逃亡することも可能。
なお、NPCの二人は小型ナイフ(1D4+DB)にヘビ人間の毒(1D8)を付与したもので基本的には対応する。魔術の使用も行うかもしれないが、GMの判断に任せる。しかし、イゴーロナクに対して有効な魔術を知っているかはわからない。
※葉草が最高火力を叩き出した場合、戦闘終了し路凪は死亡する。探索者たちは早急に葉草を止める必要もあるかもしれない。
