# 夜想曲

GM向け情報
　突如として現れた歌姫「出雲 美砂」彼女の歌声は聞いた人すべてを魅了する。彼女の母はシュブ＝ニグラスであり、その歌声は混血による祝福で得た《セイレーンの歌》による魅了である。彼女自身その事実を知らない。メジャーデビューと同時に大きなライブが決まり喜んでいる。
　そんなライブを計画したのは彼女の歌声にシュブ＝ニグラスの片鱗を見た狂信者の「芽句 流」である。彼は美砂に楽曲を提供し、それを触媒としシュブ＝ニグラスの招来を目論む。
導入
　探索者の友人の「黒戸 亜記」は「Misago」の初ライブのチケットを手に入れたと教えてくる。次の新月の夜にMisagoの初ライブ「朔の逆月(サクノサカツキ)」が行われるらしい。
　探索者たちは彼の誘いに乗ってそのライブに行くことになった。

ニュース

Misagoの歌を聴く
　新曲(芽句プロデュースの曲)を聞く場合《POW》のレギュラーで彼女の歌と対抗ロールを行う。今後、彼女の歌を聞くたびに《POW》のハード、イクストリームの順でロールを行う。失敗で1D10時間彼女の曲を聴き続けてしまう。
　既存の曲を聞く場合は《POW》のレギュラーで彼女の歌と対抗ロールを行う。失敗で1D3時間彼女の曲を聴き続けてしまう。新曲での失敗で見る以下記述の幻覚は見ない。
　これは《セイレーンの歌声》による効果で彼女の歌を聞いたとき、強制的に発動する。耳栓やイヤホン、その他方法で簡単に防げる事にする。
　また、プレイヤーたちが自主的に聞かない場合は、《幸運》の失敗などで街なかで聞くかもしれない。
　聞き惚れた者は、自身の住んでいる街に立っている時、黒雲が立ち込める。そして、周囲の人間と自身が黒雲に包まれる幻覚を見ながら歌を聴く。

ライブについて調べる
　そのライブは1人の作曲家がすべてプロデュースしたという記事が出てくる。そして、ライブ会場は自然公園の野外ステージで行われる。
　オカルト的な記事の中には木が動いたり、黒い水たまりといったものがあったりする。
Misagoに会う
　葉草はライブ会場でリハーサルを行っているMisagoに取材を取り次いでいる。
　会場につくと当たり前だがMisagoの曲を聞くことになる。《POW》のロールを行うこと。この時、聴き惚れたものがいた場合、葉草が《精神分析》を行う。
　Misagoは初ライブの意気込み等を非常にうれしそうに話してくれる。
　対人関係技能に成功したものがいれば、彼女は母がおらず父子家庭であることを教えてくれるだろう。「お父さん、私の歌が大好きなんですよ」などと教えてくれる。また、歌が好きな理由は物心つく頃にいなくなってしまった母が好きだったからということを教えられる。有名になって自分の歌声を聞けば母に会えるかもと思っている。
　彼女と会う時、芽句 流ももちろんその場に居合わせるだろう。
　彼に美砂について聞くと「愛する女性（シュブ＝ニグラス）に届けたい曲ができた。それを歌うべきは美砂しかいない」という。それは未発表でライブで初披露する曲だ。曲名は「夜想曲(ノクターン)」。
　頼み込めば弾いて聴かせてくれるかもしれない。今度のライブは彼がすべてプロデュースしている。

ライブ前日の夜
　葉草を一人にしていた場合、探索者たちの一人は葉草と出会う。彼はボロボロで気絶している。これは無形の落とし子に襲われたためである。
　彼の看病もしくは病院に連れていく場合、意識を取り戻す。
　彼は《無形の落とし子の召喚/従属》を習得している。探索者に《無形の落とし子の召喚/従属》を教えてくれる。本来1D8日かかるが1日以内で覚えることができることにする。
　それまでのロール次第で特殊な形のナイフ（1D4のダメージ、ベンズナイフ）と毒（《CON》のイクストリームに失敗で1D8のダメージ）を与える。傷口に塗るもしくは飲ませると対抗ロールが発生する。毒は無形の落とし子へ特攻となり《CON》のロールを行わずに1D8のダメージを与える。
朔の逆月
　Misagoの初ライブである。しかし、それは邪悪な思惑により《シュブ=ニグラスの招来》となっている。招来の邪魔をする方法としては『ライブジャック』や『美砂を攫う』、『芽句を倒す』などだろう。
　また、観客席には幾人か目が虚ろな人間がいる。これは病院から抜け出した「眠り山羊病」の患者たちである。黒戸もその場にいても良いかもしれない。
ライブの構成
　Misagoの顔はライティングで逆光になるようにされており基本見えない。前方の席であれば見えるかもしれないが、スクリーンに映されている映像などでは映らないようにされている。
　ライブは大きく3部に分かれている。
雲
　情景を描くような曲が多い。メインで歌われる曲名は『ひつじ雲』。
祭
　人々の営みや楽しげな曲が中心。メインで歌われる曲名は『神楽』。
シレーネ
　女性視点のかなわぬ恋の曲。メインで歌われる曲名は『セイレーンの母』。
ライブの進行
　ライブが進行するにつれ雲が夜空を覆い星が見えなくなっていく。ライブが最後まで行われるとシュブ＝ニグラスが招来されすべてが喰われる。
　主に無形の落とし子2体と芽句をどうにかすることが目的となるだろう。無形の落とし子は会場スタッフに化け紛れ込んでいる。これはライブが進むにつれ溶けだす。芽句が気絶すると無形の落とし子の使役が解け、芽句は喰われる。
　無形の落とし子はプレイヤーの技能などによっては葉草が退散済みということで1体に変更するなどしてもいいかもしれない。

戦闘勝利
　芽句が無形の落とし子に喰われるなど戦闘が終了した場合、他に警備などはいるだろうがライブの機材を触ることもできるだろう。そこで修正した音源を流せば《シュブ＝ニグラスの招来》は止まり、雲は晴れ都会の中心部とは思えないほどきれいな星空が見える。音源を止めた場合、ライブは機材トラブルにより中止となるだろう。
シュブ＝ニグラスの招来
　空から1本の太い触手が伸びてきて何人かの客を捕まえた。思わず空を見上げたものは黒雲から幾本もの触手がこちらへ伸びてきているのを見た。触手の先は蹄のようになっているものもあり、何本かは人間を簡単に踏み潰した。
　客たちは悲鳴を上げ我先に逃げ出そうとした。しかし、気づくと会場は木で囲まれていた。そして、その木は人間を踏み潰した。
　正気度喪失：1D10/1D100
