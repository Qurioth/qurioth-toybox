# 夜想曲

このシナリオは "クトゥルフ神話TRPG ルールブック 7版" に対応したシナリオで、探索者 3 ～ 5 人向けにデザインされている。  
プレイ時間は探索者の作成時間を含まずに 5 ～ 6 時間程度だろう。  
舞台は現代日本。突如として現れた歌姫「Misago」の初ライブをきっかけに、探索者たちはシュブ＝ニグラス招来を目論む狂信者の計画へ巻き込まれていく。

## GM 向け情報

突如として現れた歌姫「出雲 美砂」は、その歌声を聞いた者すべてを魅了する。  
彼女の母はシュブ＝ニグラスであり、美砂の歌声は混血による祝福として発現した《セイレーンの歌声》である。しかし、美砂自身はその事実を知らない。彼女はメジャーデビューと同時に大きなライブが決まり、純粋に喜んでいる。

このライブを計画したのは、美砂の歌声にシュブ＝ニグラスの片鱗を見た狂信者「芽句 流」である。  
芽句は美砂に楽曲を提供し、それを触媒として《シュブ＝ニグラスの招来》を目論んでいる。

シナリオの中心となる事件は以下の通りである。

- Misago の歌を聞いた人々が魅了され、「眠り山羊病」と呼ばれる催眠状態に陥る。
- 地域では行方不明者が続出している。
- 行方不明者や生き物は、招来儀式の供物として無形の落とし子によって集められている。
- 探索者は、葉草 夏月の協力、芽句の家の調査、音源の修正、ライブ当日の妨害などを通じて招来を阻止する。

## シナリオの導入

探索者の友人である「黒戸 亜記」は、「Misago」の初ライブのチケットを手に入れたと探索者たちに教えてくる。  
次の新月の夜、Misago の初ライブ「朔の逆月（サクノサカツキ）」が行われるらしい。

黒戸は Misago の歌に強く惹かれており、探索者たちにその魅力を熱心に語る。探索者たちは彼の誘いに乗り、そのライブに行くことになる。

## Misago について

黒戸やネットから、Misago について以下の情報が得られる。

- 動画サイトに「歌ってみた」動画を投稿している。
- 破竹の勢いで再生数とファンを獲得している。
- 彼女は一切顔出しをしていない。
- 歌っている曲は、特別最近の曲でも、有名な曲でもない。
- それにもかかわらず、彼女の歌ってみた動画は高い再生数を稼いでいる。
- ネット上では、基本的に彼女を称賛する記事や投稿ばかりが目につく。

## 黒戸の入院

ライブの 3 日前、黒戸が入院する。病院は東京都立豊島病院など、探索者が訪れやすい場所に設定するとよい。

黒戸は虚ろな目で天井を見つめている。  
そして、ブツブツとうわ言のように何かを呟いている。

**〈聞き耳〉** などに成功した場合、探索者は以下の言葉を聞き取る。

```
千匹の仔山羊を連れし黒雲、我が子の歌声により今舞い降りん
```

この言葉を聞いた探索者は、**正気度喪失：0 ／ 1**。

黒戸の持ち物を調べた場合、葉草 夏月というジャーナリストの名刺が出てくる。  
探索者が情報不足で行き詰まりそうな場合、葉草の方から訪ねてきてもよい。または、黒戸の携帯電話に葉草から着信が入る。

### 病院で分かること

黒戸と同じ症状の患者が多く入院している。  
彼らは Misago の歌を聞き、魅了された結果として発症している。病院関係者内では、この症状は「眠り山羊病」と呼ばれており、一種の催眠状態に陥っていると考えられている。

## ニュースと行方不明事件

ニュースでは、最近この地域で行方不明者が続出していることが取り上げられている。

被害者同士に関わりはなく、共通点も見つかっていない。親族などに話を聞いても、不審な点はない。

証言例は以下の通り。

> 「ちょっと散歩に出かけたきり、帰ってこないんです」  
> 「普段通り仕事に出かけて行っただけでした」  
> 「特に変わった様子はありませんでした」

この事件の背景には、《シュブ＝ニグラスの招来》のために、芽句の命令を受けた無形の落とし子が生き物を集めていることがある。

**〈アイデア〉** などに成功した場合、行方不明者が最後に目撃された地点の中心付近に、ライブ会場が位置していることに気づく。

## Misago の歌を聴く

Misago の歌を聞いた探索者は、《セイレーンの歌声》の影響を受ける。  
この効果は彼女の歌を聞いたときに強制的に発動する。

### 新曲を聞く場合

芽句がプロデュースした新曲を聞いた場合、**POW のレギュラー難度** で彼女の歌と対抗ロールを行う。  
以降、彼女の歌を聞くたびに、難度は以下のように上昇する。

1. レギュラー
2. ハード
3. イクストリーム

失敗した場合、探索者は **1D10 時間**、彼女の曲を聴き続けてしまう。

### 既存曲を聞く場合

既存の曲を聞いた場合、**POW のレギュラー難度** で彼女の歌と対抗ロールを行う。  
失敗した場合、探索者は **1D3 時間**、彼女の曲を聴き続けてしまう。

既存曲で失敗した場合、新曲で見る幻覚は発生しない。

### 防御手段

耳栓、イヤホン、その他の方法により、歌を直接聞かないようにすれば簡単に防げることにする。

探索者が自主的に Misago の歌を聞かない場合でも、**〈幸運〉** の失敗などにより、街中で偶然聞いてしまう可能性がある。

### 幻覚

聞き惚れた探索者は、自分の住んでいる街に立っている幻覚を見る。  
空には黒雲が立ち込め、周囲の人間と自分自身が黒雲に包まれていく。探索者はその幻覚の中で、Misago の歌を聴き続ける。

**〈聞き耳〉** に成功した場合、以下の声が聞こえる。

```
呼ぶ声が聞こえる。
呼ばれている……。
懐かしい響き……、ああ、会える。
```

その声は男性とも女性ともつかない不思議な声で、いくつもの言語で同時に発されている。  
**正気度喪失：1 ／ 1D6**

**〈目星〉** に成功した場合、黒い煙のような黒雲から、黒い触手が絶えずうねり続けているのが見える。  
そして、その一本が探索者の身体に触れた気がする。  
**正気度喪失：1 ／ 1D6**

《セイレーンの歌声》の効果が切れた後、自分の意志に反して行動していたことを理解した探索者は、**正気度喪失：0 ／ 1D3**。

## ライブについて調べる

ライブについて調べると、以下の情報が得られる。

- ライブは 1 人の作曲家がすべてプロデュースしている。
- ライブ会場は自然公園の野外ステージである。
- オカルト系の記事には、会場周辺で「木が動いた」「黒い水たまりを見た」といった噂がある。

## Misago に会う

葉草は、ライブ会場でリハーサルを行っている Misago に取材を取り次いでいる。  
探索者が会場に到着すると、Misago の曲を聞くことになるため、**POW ロール** を行うこと。

この時、聴き惚れた探索者がいた場合、葉草が **〈精神分析〉** を行う。

Misago は、初ライブの意気込みなどを非常に嬉しそうに話してくれる。  
**対人関係技能** に成功した探索者がいれば、彼女は以下のようなことを教えてくれる。

- 母がおらず、父子家庭である。
- 父は彼女の歌が大好きである。
- 歌が好きな理由は、物心つく頃にいなくなった母が歌を好きだったからである。
- 有名になれば、自分の歌声を聞いた母に会えるかもしれないと思っている。

> 「お父さん、私の歌が大好きなんですよ」  
> 「もしかしたら、有名になったら……お母さんにも届くかもしれないじゃないですか」

美砂と会う時、芽句 流もその場に居合わせる。  
芽句に美砂について聞くと、以下のように語る。

> 「愛する女性に届けたい曲ができた。それを歌うべきは、美砂しかいない」

ここで芽句が言う「愛する女性」とは、シュブ＝ニグラスを指している。  
その曲は未発表であり、ライブで初披露される。曲名は「夜想曲（ノクターン）」である。

頼み込めば、芽句はその曲を弾いて聴かせてくれるかもしれない。  
今回のライブは、彼がすべてプロデュースしている。

## 取材終了後

取材終了後、葉草は行き先を告げずに「ついてこい」と探索者たちに言う。  
彼についていくと、芽句の家に到着する。

葉草が一言二言、何かを紡ぐと、芽句の家の鍵が開く。

## 芽句の家

芽句の家は、2LDK の一階建ての戸建てである。

家に着いた時点で、探索者は **〈目星〉** や **〈聞き耳〉** などを行ってもよい。  
成功した場合、「無形の落とし子」の存在を感知する。これは、取材完了時に芽句が探索者たちを不審に感じ、後をつけさせているためである。

## ライブ会場

ライブ会場は、野外ステージと、その裏に広がる雑木林を持つ自然公園である。  
普段は散歩やランニングをする人が多くいる。

何らかの技能に成功した探索者は、違和感を覚える。  
公園内には、人間以外の生き物の気配がない。犬の散歩をしている人もいるが、犬たちは公園に入ることを嫌がっているか、急いで公園から出ようとしている。

### 野外ステージ

3000 人は入る会場である。  
普段は有名アーティストなどがライブを行っている。

### 雑木林

裏の雑木林に入ると、何かを引きずったような跡と、点々と黒いペンキのようなものを見つける。  
**〈医学〉** に成功した探索者は、それが黒く変色した血であると理解する。血とは違う黒い粘着質な液体も付着している。

奥に進むと、黒い女性の像――ヤスナ・グラの聖母――が建っている。  
鉄臭いにおいと、何かが腐敗したにおいが漂っている。

調べると、10 数人ほどの人間の死体と、様々な生き物の死体を見つける。死体の総量は 1000 SIZ 程度である。  
**正気度喪失：1D3 ／ 1D6+1**

これは、シュブ＝ニグラスを招来するために芽句が集めたものである。  
また、周辺には黒い仔山羊がいる。

## 美砂の母の捜索

探索者が美砂の母を探し始めた場合、自分を探している人間がいると知ったシュブ＝ニグラスは、探索者に興味を持って寄ってくるかもしれない。

登場例は以下の通り。

- 美砂とよく似た女性を街中で見かける。
- 自然公園に、非常に綺麗な女性がいる。
- TV の街頭インタビューに映っている。
- 街中でナンパされている。

この女性が、守部 保奈を名乗るシュブ＝ニグラスである。

## ライブ前日の夜

葉草を一人にしていた場合、探索者の一人は葉草と出会う。  
彼はボロボロで気絶している。これは、無形の落とし子に襲われたためである。

探索者が彼を看病する、または病院に連れていく場合、葉草は意識を取り戻す。

葉草は《無形の落とし子の召喚／従属》を習得している。  
彼は探索者に《無形の落とし子の召喚／従属》を教えてくれる。本来は 1D8 日かかるが、このシナリオでは 1 日以内で覚えることができるものとする。

それまでのロール次第で、葉草は探索者に以下を渡す。

- 特殊な形のナイフ
  - ダメージ: 1D4
  - ベンズナイフとして扱う
- 毒
  - 通常: **CON のイクストリーム** に失敗で 1D8 ダメージ
  - 無形の落とし子に対して: CON ロールなしで 1D8 ダメージ
  - 傷口に塗る、または飲ませることで対抗ロールが発生する

## クライマックス

### 朔の逆月

Misago の初ライブ「朔の逆月」が始まる。  
しかし、このライブは邪悪な思惑により、《シュブ＝ニグラスの招来》として機能している。

招来を邪魔する方法としては、以下が考えられる。

- ライブジャックを行う。
- 美砂を攫う。
- 芽句を倒す。
- 修正した音源を流す。
- 無形の落とし子の使役を奪う、または退散させる。

観客席には、幾人か目が虚ろな人間がいる。  
これは、病院から抜け出した「眠り山羊病」の患者たちである。黒戸もその場にいてよい。

### ライブの構成

Misago の顔は、ライティングで逆光になるように調整されており、基本的には見えない。  
前方の席であれば直接見えるかもしれないが、スクリーンに映される映像では、彼女の顔は映らないようにされている。

ライブは大きく 3 部に分かれている。

#### 雲

情景を描くような曲が多い。  
メインで歌われる曲名は『ひつじ雲』。

#### 祭

人々の営みや、楽しげな曲が中心。  
メインで歌われる曲名は『神楽』。

#### シレーネ

女性視点の、かなわぬ恋の曲。  
メインで歌われる曲名は『セイレーンの母』。

### ライブの進行

ライブが進行するにつれ、雲が夜空を覆い、星が見えなくなっていく。  
ライブが最後まで行われると、シュブ＝ニグラスが招来され、すべてが喰われる。

主な障害は、芽句 流と無形の落とし子 2 体である。  
無形の落とし子は会場スタッフに化けて紛れ込んでおり、ライブが進むにつれて溶け出す。

芽句が気絶すると、無形の落とし子の使役が解ける。その結果、芽句は無形の落とし子に喰われる。

探索者の行動や技能次第では、葉草がすでに 1 体を退散させていたことにして、無形の落とし子を 1 体に変更してもよい。

### 芽句 流 ＆ 無形の落とし子との戦闘

| ターン     | 芽句 流の行動                            | 無形の落とし子の行動                             |
| ---------- | ---------------------------------------- | ------------------------------------------------ |
| 1 ターン目 | 無形の落とし子の使役                     | DEX 順では芽句より前だが、このターンは行動しない |
| 2 ターン目 | 詠唱                                     | ランダムな対象を攻撃                             |
| 3 ターン目 | 詠唱                                     | ランダムな対象を攻撃                             |
| 4 ターン目 | 膝をつき、回避を行わなくなる。詠唱は継続 | ランダムな対象を攻撃                             |
| 5 ターン目 | 詠唱                                     | ランダムな対象を攻撃                             |
| 6 ターン目 | 訪れない                                 | シュブ＝ニグラスが招来される                     |

### 戦闘勝利

芽句が無形の落とし子に喰われるなどして戦闘が終了した場合、探索者はライブ機材を操作できる可能性がある。  
修正した音源を流せば、《シュブ＝ニグラスの招来》は止まる。雲は晴れ、都会の中心部とは思えないほど綺麗な星空が見える。

音源を止めた場合、ライブは機材トラブルにより中止となる。

### シュブ＝ニグラスの招来

招来が完了した場合、以下の描写を行う。

```
空から、一本の太い触手が伸びてきた。

それは観客席へ降り注ぎ、何人かの客をまとめて捕まえる。
思わず空を見上げた者は、黒雲の奥から幾本もの触手がこちらへ伸びてきているのを見る。

触手の先は、蹄のようになっているものもある。
何本かは、逃げ惑う人間をあまりにも簡単に踏み潰した。

客たちは悲鳴を上げ、我先に逃げ出そうとする。
だが、気づけば会場は木々に囲まれていた。

そして、その木々もまた、人間を踏み潰した。
```

この光景を目撃した探索者は、**正気度喪失：1D10 ／ 1D100**。
